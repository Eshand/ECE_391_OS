Operating System created in the class ECE 391 at the University of Illinois at Urbana-Champaign. 
Partners are Pavan Baloo, Kshitij Sharma, and Dilan Patel. 

A EXT2 based core Linux OS with an interrupt descriptor table (IDT), basic paging support for tasks, separate 4 MB pages for the kernel and applications, initialized devices (RTC, PIC, PIT, keyboard), system call interface (including assembly linkage and C handlers), support for six concurrent tasks from program images in the filesystem, implementation of multiple simultaneous terminals, and Round Robin Scheduling. Executables implemented include ls, grep, cat, and more.


Must be run on Illinois specific virtual machine.
